Welcome to Ablurb

Amazon KDP is an amazing service which allows you to publish eBooks without much of a hassle.

Unfortunately, there's currently no feature available to preview the blurb you input for your book.

I whipped together a quick tool to allow you to preview the changes — Ablurb!

Type in the textarea below to get started!

If you have a feature request, want to report a bug or just say hi, my Reddit name's MakingTypingNoises.
